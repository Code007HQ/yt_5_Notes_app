# Frontend - React + Vite + TypeScript

This is the frontend for the project, built using **Vite, React, and TypeScript**.

## 🚀 Live Demo  
[Deploy Link - Notes App](https://yt-5-notes-app.vercel.app/)

## 🛠️ Setup Instructions

### 1️⃣ Clone the Repository
```sh
git clone https://github.com/Code007HQ/yt_5_Notes_app.git
cd yt_5_Notes_app
```

### 2️⃣ Install Dependencies
```sh
pnpm install
```
### 4️⃣ Start the Development Server
```sh
pnpm dev
```

The app should now be running at http://localhost:5173

🛠️ Tech Stack
+ React (Vite + TypeScript)
+ Tailwind CSS (For styling)
+ Axios (For API requests)
+ Shadcn UI Library
